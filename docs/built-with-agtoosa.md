# Built with AgToosa

> **Claim Boundary:** This page is a **curated** index of external examples.
> Listings are **not** paid placement, **not** official AgToosa certification,
> and **not** a security or quality endorsement of any third-party project.

Visitors can evaluate AgToosa in real repositories. Contributors can propose a
listing with clear evidence. Maintainers curate manually — empty at launch is
valid.

## Eligibility

### Inclusion criteria

A project may be considered when all of the following hold:

- **Public repository** with a stable URL maintainers can review
- **Verifiable AgToosa usage** (install marker, workflow docs, or equivalent proof)
- Honest description that does not overstate AgToosa endorsement
- No undisclosed malware, abuse, or deceptive packaging

### Exclusion criteria

Do **not** list projects that:

- Make **misleading** claims about AgToosa certification or endorsement
- Appear to be scraped **content farms** or bulk SEO shells
- Host or promote **malware** / **abuse**
- Lack a public repo or refuse a maintainer contact path

## Submission rules

Propose a listing via a GitHub issue or pull request against this repository.
Required evidence:

| Field | Required |
|-------|----------|
| Public repository link | Yes |
| Short description (1–3 sentences) | Yes |
| AgToosa version or proof reference | Yes (e.g. `Docs/.agtoosa-version`, commit, or verify output pointer) |
| Contact / issue path for maintainer review | Yes (issue URL, PR, or maintainer handle) |

Optional: link to a voluntary case-study write-up (see below). Logo-only or
marketing-only submissions without a repo link are rejected.

### Rejection reasons

- Missing required fields
- Unverifiable AgToosa usage
- Forbidden claim language (see below)
- Out of scope (private repo, paid placement request, hosted binaries)

## Forbidden claim language

Do **not** use language that implies official endorsement or security
certification of listed projects. Examples of **forbidden** phrasing:

- “AgToosa certified” / “AgToosa Certified Projects”
- “AgToosa security approved” / “security certified by AgToosa”
- “officially endorsed by AgToosa”
- Paid placement framed as editorial selection

Allowed framing: “curated example,” “uses AgToosa,” “community listing,”
“maintainer-reviewed for showcase eligibility only.”
