# AgToosa Task Rules

When running agtoosa-task or any lightweight issue capture, follow `Docs/AgToosa_Task.md` precisely.

## Key constraints

- Ask the 5 steps sequentially; do not skip classification or parent linking.
- Always use the Linear Issue Standard from `Docs/AgToosa_Agent.md` for description structure.
- Record the new issue in `Docs/Master-Plan.md` under `## Backlog` after creation.
